LongCat v2 for Copilot Chat 使用你自己的 LongCat API Key，让 LongCat 2.0 出现在模型选择器中。

只需粘贴一次，之后可通过命令面板更新或移除。

- `Cmd/Ctrl + Shift + P`：打开命令面板
- `LongCat: 设置 API Key`：设置或更新 API Key
- `LongCat: 清除 API Key`：移除 API Key
- `LongCat: 获取 API Key`：在 [longcat.chat/platform/api_keys](https://longcat.chat/platform/api_keys) 创建 LongCat API Key
