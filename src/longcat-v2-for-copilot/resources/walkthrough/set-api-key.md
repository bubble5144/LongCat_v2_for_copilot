LongCat v2 for Copilot Chat uses your own LongCat API Key to make LongCat 2.0 available in the model picker.

Paste it once, then update or remove it later from the Command Palette.

- `Cmd/Ctrl + Shift + P`: Open the Command Palette
- `LongCat: Set API Key`: Set or update your API Key
- `LongCat: Clear API Key`: Remove your API Key
- `LongCat: Get API Key`: Create a LongCat API Key at [longcat.chat/platform/api_keys](https://longcat.chat/platform/api_keys)
