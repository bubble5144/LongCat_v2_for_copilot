配置好 API Key 后，LongCat 2.0 会出现在 Copilot Chat 模型选择器中。

1. 打开 Copilot Chat（`Ctrl+Shift+I` / `Cmd+Shift+I`）
2. 点击聊天输入框上方的模型下拉菜单
3. 选择 **LongCat 2.0**

支持的模型能力：
- **思考模式** — 在模型选择器的内联配置中切换
- **工具调用** — 完整 Agent 模式（文件编辑、终端、搜索、Git、MCP、Skills）
- **1M token 上下文** — 处理大型代码库和长对话
- **视觉代理** — 将图片拖入聊天，由视觉模型描述后以文本形式发送给 LongCat
