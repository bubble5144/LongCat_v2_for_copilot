Once your API Key is configured, LongCat 2.0 appears in the Copilot Chat model picker.

1. Open Copilot Chat (`Ctrl+Shift+I` / `Cmd+Shift+I`)
2. Click the model dropdown in the chat input area
3. Select **LongCat 2.0**

The model supports:
- **Thinking mode** — toggle in the model picker's inline configuration
- **Tool calling** — full Agent mode (file edits, terminal, search, Git, MCP, skills)
- **1M token context** — handle large codebases and long conversations
- **Vision proxy** — drop images into chat; they are described by a vision model and fed to LongCat as text
