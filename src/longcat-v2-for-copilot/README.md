# LongCat v2 for Copilot Chat

在 Copilot Chat 模型选择器中使用美团 LongCat 2.0——保留 Copilot 全部能力（Agent 模式、工具调用、Skills、Instructions、MCP），零配置、自带 API Key、无需额外进程。

> Pick LongCat 2.0 from the Copilot Chat model picker — keep every Copilot capability (agent mode, tools, skills, instructions, MCP) running on Meituan's LongCat 2.0. Zero config, BYOK, no extra process.

---

## 快速开始

1. 在 [longcat.chat/platform/api_keys](https://longcat.chat/platform/api_keys) 创建 API Key
2. 在 VS Code 命令面板运行 **LongCat: Set API Key**，粘贴 Key
3. 打开 Copilot Chat，在模型选择器中选择 **LongCat 2.0**

## 功能

- **Agent 模式** — 文件编辑、终端、搜索、Git，全部原生
- **思考模式** — 模型选择器中切换
- **1M token 上下文** — 处理大型代码库
- **视觉代理** — 图片拖入聊天，自动转文字描述
- **Token 用量追踪** — 状态栏实时显示
